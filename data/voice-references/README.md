# Voice-Stylometric Reference Corpus — Passages

Prose for the Stage 3 voice agents' stylometry tool. Metadata for each entry
lives in `data/voice-references-stylometric.yaml`; the actual prose lives
here as `<id>.txt` files.

The split keeps git diffs small and lets the corpus grow without code
changes.

## Provenance

dataset (`lab/scratch/tribe-prose-pilot/corpus_v2/` and `corpus_v3_llm/`).

## Acknowledged gaps

- **Contemporary literary minimalist/maximalist** (Carver, McCarthy,
  Pynchon, DFW, Hempel, Davis, Strout) — no public corpus exists; copyright
  intact. Phase 2 of `2026-04-28-corpus-author-gaps.md` ships the manual
  fair-use staging path (`lab/fair-use-staging/`); user-paced curation when
  ready. Hemingway iceberg-register and Faulkner stream-of-consciousness
  now both anchored via round 6.
- **Free indirect discourse-annotated English corpus** doesn't exist
  publicly (closest analog: the German Brunner et al. 2019). Hand-tagging
  passages from Joyce-Portrait / Lawrence / Mansfield / Forster (all
  in the corpus) is the workaround if FID becomes a load-bearing signal.
- **Literary journalism / longform narrative nonfiction** — no purpose-
  built corpus; All-the-News-2.0 is the nearest approximation but skews
  news-reporting, not literary.

## Stratification

The reference picker uses tags to return a balanced mix per tool call.
Recommended composition spans:

- 1 `llm_default` (DeepSeek literary baseline — what voice candidates
  measure distinctiveness against)
- 1 `expository` (Wikipedia neutral-register baseline)
- 1 `human_amateur` (unpolished human prose anchor)
- 2 `literary` (one minimalist/restrained, one maximalist/digressive)

## Adding a new reference

1. Append an entry to `data/voice-references-stylometric.yaml` with a unique
   `id`, `tags`, `source`, `license`, and `text_file: passages/<id>.txt`.
2. Drop the prose into `passages/<id>.txt`.
3. The signal cache (`data/voice-references-stylometric.cache.json`) rebuilds
   automatically on next tool load when the corpus YAML's content hash
   changes; or run `uv run python -m owtn.tools rebuild-cache` to force
   a refresh.

## Encoding

UTF-8. Preserve paragraph breaks as blank lines. Do not hard-wrap lines
inside a paragraph.

## What if a passage file is missing?

The loader logs a warning and skips that entry. The tool remains usable
with whatever subset of the corpus is populated.

## License posture

Public-domain entries (canonical literary, PG19) and CC-BY-SA entries
(Wikipedia simple) are included freely. CDLA-Sharing-1.0 (TinyStories) and
research-fair-use (WritingPrompts) entries are included under the
project's research-not-product posture (see
`project_research_not_product.md` in memory). DeepSeek pilot generations
are the user's own outputs from a research pilot (license:
`owtn_pilot_generation`).
